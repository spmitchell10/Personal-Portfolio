About

This is a SoundCloud recreation that was built in one day on week three, day 3 during my time at The Iron Yard. This website utilizes AJAX to make a request using the Sound Cloud for Developers. This was the first web application that I used the audio tag. The web app also utilizes parallax scrolling for longer music lists. Plug in your favorite artist and listen away!
