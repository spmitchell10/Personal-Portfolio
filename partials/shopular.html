MARVEL FAN SITE

About

This awesome Marvel fan page was build in one day on week three, day four during my time at The Iron Yard. This website utilizes AJAX to make a request using the Marvel Developers API. This website also integrates a google map using Google Maps API.

As an added feature, each character and comic has a 'learn more' feature that pops up in a modal to display certain traits of a character and a list of comics they appear in. This feature was added using Bootstrap.
