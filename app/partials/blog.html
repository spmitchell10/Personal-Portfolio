About

This was the final weekend project during my time at The Iron Yard, completed in week 9. This project utilized my skills on the front end, as well as a fully built out back-end using NodeJS. This project utilizes MongoDB for storage and has a secure admin page for posting new Blogs.

The home page displays the top 10 most recent blog posts, limiting the content to only 300 letters. If you would like to read more, simply click the "Read More" button, which takes you to the full post.
