SHOPULAR ADMIN PANEL

About

Shopular was built as my first AngularJS application throughout three days during week six at The Iron Yard for Front-End Engineering. This application is an admin product listing page which displays product information in a table format.

When you enter the application, the admin has the option of adding a new item to the table. Simply click the button, "Add a Product!" in the top left corner and a modal form pops up to add a new item to the list.

All information for this application is persisted using the browsers localstorage.
